Pen:

Pen::Pen(float r, float g, float b){

   

 

    red=r;

    green=g;

 

    blue=b;//initialize the color

 

     m_mask = new float *[3]; //initialize the mask

    for(int i=0;i<3;i++){

        m_mask[i]=new float [3];

    }

    m_mask[0][1]=1.0;//set the shape cross

m_mask[1][0]=1.0;

m_mask[1][1]=1.0;

m_mask[1][2]=1.0;

m_mask[2][1]=1.0;

}

 

void Pen::setColor(float r, float g, float b){//change the color

    red=r;

    green=g;

    blue=b;

}

 

void Pen::draw(PixelBuffer *background, int x, int y){

 

    for(int i=0;i<3;i++){//draw on panel

        for(int c=0;c<3;c++){

            if(m_mask[i][c]!=0){

            const ColorData& color=ColorData(red,green,blue);

            background->setPixel(x-1+c,y-1+i,color);

             }

        }

    }

 

}

Pen::~Pen(){

    if(m_mask){

        for(int i=0;i<3;i++){

 

           delete m_mask[i];

        }

        delete m_mask;

    }

}

 

This snippet is from the Pen class, which is one of the two classes I wrote for our team.

The purpose of this section of code is to simulate a Pen for this drawing project. So there are three useful functions. The first is the constructor, when this Pen is being initialized, the color from the Panel will be pass to this Pen and set pen¡¯s color to the color, then a 3*3 mask is initialized, and the shape will be a cross. The draw() is to draw on the Panel. In this function, the Panel and the location which the user click on will be passed into. Then we just go through the mask, if the value is equal to one, we just find the corresponding location on the Panel and replace the pixel by the Pen¡¯s color.

The setColor() is to change the Pen¡¯s color, when the user changes color in main user interface, this function will get called and the color the user chose will be passed.

 

SprayCan:

SprayCan::SprayCan(float r, float g, float b){

    red=r;

    green=g;

    blue=b;//initialize the color

 

     m_mask = new float *[41];//initialize the mask

    for(int i=0;i<41;i++){

        m_mask[i]=new float [41];

    for(int c=0;c<41;c++){//set the shape circle

          float d=1-0.05*(int)(sqrt((20-i)*(20-i)+(20-c)*(20-c)));

          if(d<0){

              d=0;

          }

          m_mask[i][c]=d;

    }

    }

 

}

 

void SprayCan::setColor(float r, float g, float b){//change the color

    red=r;

    green=g;

    blue=b;

}

 

void SprayCan::draw(PixelBuffer *background, int x, int y){

 

    for(int i=0;i<41;i++){//draw on panel

        for(int c=0;c<41;c++){

            if(m_mask[i][c]>0){

      ColorData color=background->getPixel(x-20+c,y-20+i);

      float influ=m_mask[i][c]*0.2;

      const ColorData& colo=ColorData(influ*red+(1-influ)*color.getRed(),influ*green+(1-influ)*color.getGreen(),influ*blue+(1-influ)*color.getBlue());

            background->setPixel(x-20+c,y-20+i,colo);

}

        }

    }

}

SprayCan::~SprayCan(){

    if(m_mask){

        for(int i=0;i<41;i++){

 

           delete m_mask[i];

        }

        delete m_mask;

    }

}

 

This snippet is from the SprayCan class, which is one of the two classes I wrote for our team.

The purpose of this section of code is to simulate a SprayCan for this drawing project. So there are three useful functions. The first is the constructor, when this SprayCan is being initialized, the color from the Panel will be pass to this Pen and set pen¡¯s color to the color, then a 41*41 mask is initialized, and the shape will be a circle whose radius is 20, and it will decrease linearly with the increase of radius. For example, the center of the mask is 20,20, whose value is 1, and the pixel which is 20 away from the center is 0. The draw() is to draw on the Panel. In this function, the Panel and the location which the user click on will be passed into. Then we just go through the mask, if the value is greater than one, we just find the corresponding location on the Panel and create a new ColorData which is made 40%*mask value of the tool¡¯s color and 60% of the original color.

The setColor() is to change the Pen¡¯s color, when the user changes color in main user interface, this function will get called and the color the user chose will be passed.

 

 

Code style:

The code style for this project is simulate the real life as much as possible. For example, in real life, the pen will have shape which is the mask, color and it can draw on paper. So in code, to achieve same function, it will be better to be close to real life. Another principles are using 2d array, using for loop not while loop, making comments when necessary. These principles will make other team members easily understand the code and make the code be easily connected with other members¡¯ code.
